Mandelbrot Fractals

In 2k17 I've chosen as task in POSIX C to realize a Pixel Generator which exchanges data over shared memory and coordinates the synchonisatzion between a generator process and a receiver process with semaphores.

As one proposal, the generator should calculate the Mandelbrot set in a loop. The calculation itself, thanks to thousands of examples found in internet, was easy to realize, much easier than the interprocess communication.

The task was not the Mandelbrot set calculation, but this got for me to the most fascinating part of this task. I started to work and experiment with different color mappings and spent lot of kwh in electrical power.

Due to that reason I recoded after handing in my task the whole tool to a standalone, 4 thread application for Linux. I will continue working on that stuff, planning (already started) a nice GTK-3 GUI with support for mouse actions, save as JPG possibility, more and intelligent calculation threads and smooth fancy color mapping. Check out mit github respositority, download the code (LINUX), compile and have fun...
